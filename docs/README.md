## üì¶ Project Information

### What This App Does

The DevFest Competition Form is a complete platform for:

1. **AI Innovation Lab Competition**
   - Project submissions with screenshots
   - Public gallery
   - Admin panel for winner selection
   - Draft system

2. **AI Devcamp 2026 Mentorship Program**
   - Mentee applications
   - Mentor applications with resume upload
   - Tag-based skill matching
   - Application tracking

### Tech Stack

- **Framework**: Next.js 14.2 (App Router)
- **Language**: TypeScript 5.6
- **Authentication**: Firebase Auth (Email/Password + Google)
- **Database**: Firebase Firestore
- **Storage**: Firebase Storage
- **UI**: shadcn/ui + Tailwind CSS
- **Deployment**: Vercel

### Key Features

- üîê Firebase Authentication (no custom domain needed)
- üìù Draft system for submissions
- üì∏ Multi-screenshot upload (up to 5)
- üèÜ Winner selection (1st, 2nd, 3rd place)
- üë• User role management (Admin, Moderator, User)
- üéì Mentorship program with applications
- üè∑Ô∏è Smart tag system for skills
- üìÑ Resume upload for mentors
- üì± Fully responsive design
